# Plan 01-02: Config Watcher Thread

**Objective:** Add a low-latency file watcher in `opensim_live_realtime.py` that loads the joint display filter from the sidecar JSON without blocking the ~20 Hz visualizer loop.

**Requirements:** DISP-03

---

<task id="T1" type="execute">
<title>Add joint display config state and watcher thread</title>

<read_first>
- opensim_live_realtime.py (full file — threading, _load_sensor_map, run_live loop)
- opensim_joint_catalog.py
- docs/opensim-joint-display-config.md
- .planning/phases/01-display-config-contract-spike/01-RESEARCH.md (watcher design)
</read_first>

<action>
In `opensim_live_realtime.py`:

1. Add constant `JOINT_DISPLAY_CONFIG = "opensim_joint_display_config.json"`

2. Add module-level thread-safe state:
   - `_display_filter_lock = threading.Lock()`
   - `_display_filter_joints = []` (validated coordinate names)
   - `_display_filter_seq = -1`

3. Implement `_read_joint_display_config()`:
   - Path: `os.path.join(WORK_DIR, JOINT_DISPLAY_CONFIG)`
   - Parse JSON; require `joints` (list), `seq` (int)
   - Use `opensim_joint_catalog.validate_joints()` on joints list
   - Return `(seq, validated_joints)` or `None` on missing/malformed file

4. Implement `_joint_display_watcher_thread()` daemon:
   - Poll interval 0.05 s (50 ms)
   - Track last mtime; on change call `_read_joint_display_config()`
   - If `seq > _display_filter_seq`, update shared state under lock and print `[JOINT-DISPLAY] seq={seq} joints={names}`

5. Add `get_display_filter_joints()` returning copy of current list under lock

6. Start watcher thread in `run_live()` after UDP thread start, before main render loop

7. Do NOT modify UDP receive/send logic or packet parsing
</action>

<acceptance_criteria>
- Function `_joint_display_watcher_thread` exists in opensim_live_realtime.py
- Watcher uses 50 ms sleep interval (0.05)
- Log prefix `[JOINT-DISPLAY]` appears in watcher update path
- `get_display_filter_joints()` returns list without raising when config absent
</acceptance_criteria>

<verify>
rg -q "_joint_display_watcher" opensim_live_realtime.py && rg -q "JOINT-DISPLAY" opensim_live_realtime.py && rg -q "opensim_joint_catalog" opensim_live_realtime.py
</verify>
</task>
