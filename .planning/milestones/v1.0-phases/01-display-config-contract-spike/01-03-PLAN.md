5. Call `_render_joint_display_hud` once per rendered frame in main loop, after `ikSolver.assemble(state)` and before `model.getVisualizer().show(state)`

6. Throttle window title updates: only call setWindowTitle when HUD text string changes (avoid redundant OS calls)
</action>
